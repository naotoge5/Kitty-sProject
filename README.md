完成形をmasterブランチとする
開発段階ではmainブランチにmergeする。

               
- master
  - main
    - feature/yoshi
    - feature/imai
    - feature/tauchi
    - feature/tani
    - feature/kobayashi

ブランチの一覧確認
> git branch -a
```
  master
* main
  feature/名前
  remotes/origin/HEAD -> origin/master
  remotes/origin/master
  remotes/origin/feature/名前
```
ブランチの削除
> git branch -d feature/名前
